# Medical Insurance Cost Analysis

## Project Overview
This project analyzes medical insurance costs using Microsoft Excel. The objective is to identify factors associated with higher insurance expenses and present the findings through a dashboard.

## Dataset
- Dataset Size: 1,338 records
- Variables analyzed:
  - Age
  - Smoking Status
  - BMI
  - Insurance Charges

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## Analysis Process
1. Cleaned and reviewed the dataset.
2. Created Age Group and BMI Group categories.
3. Built Pivot Tables to summarize insurance costs.
4. Calculated average charges across different groups.
5. Developed a dashboard to visualize results.

## Key Insights
- Smoking has the strongest impact on insurance costs.
- Insurance costs increase consistently across age groups.
- Obesity is associated with higher average insurance costs.
- Older smokers have the highest average insurance costs.
